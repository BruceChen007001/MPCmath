\# 安全多方计算数学基础



\## 书籍简介

《安全多方计算数学基础》是一本旨在为读者系统构建理解安全多方计算（Secure Multi-Party Computation, SMPC）所需的数学基础的中文图书。本书通过梳理计算复杂性理论、初等数论、抽象代数与密码学等核心数学知识，为读者深入学习和研究现代密码学与安全多方计算协议奠定坚实的理论根基。



\## 内容结构

本书共分为六章，结构如下：



\### 第一章：概述

介绍安全多方计算的基本概念、核心问题与现实应用，为全书内容提供背景与脉络。



\### 第二章：计算复杂性理论基础

讲解算法复杂性的基本概念（如P、NP、NP完全问题）、计算模型（如图灵机）以及归约方法，这些是理解密码协议安全性和效率的理论框架。



\### 第三章：初等数论基础

涵盖整除理论、同余、模运算、素数、欧几里得算法、中国剩余定理、原根与离散对数等数论核心知识，这是构建多数公钥密码与安全计算协议的算术基础。



\### 第四章：抽象代数基础

介绍群、环、域、有限域等代数结构的基本概念与性质，为学习基于代数困难问题的密码体制（如椭圆曲线密码）提供必要的数学工具。



\### 第五章：密码学基础知识

系统讲解现代密码学的核心组件，包括对称密码、公钥密码、哈希函数、数字签名、消息认证码以及密码协议的安全模型（如语义安全等）。



\### 第六章：特定工具与运算

深入介绍在安全多方计算中常用的具体工具与运算，包括：

\- \*\*汉明距离与汉明重量\*\*：度量信息差异的基本工具。

\- \*\*等重码\*\*：在编码理论与安全计算中具有特殊应用的码字。

\- \*\*相等运算符\*\*：实现隐私保护的相等性比较运算，是安全多方计算中的基础操作符。



\## 目标读者

\- 密码学、信息安全、计算机科学等相关专业的高年级本科生与研究生。

\- 从事密码学理论、安全协议设计、隐私计算等领域的科研人员与工程师。

\- 希望系统构建安全多方计算数学背景的任何技术爱好者。



\## 先修知识

阅读本书需要具备离散数学、线性代数以及一定的编程基础。对算法有基本了解将有助于理解。



\## 使用说明

本书可作为“安全多方计算”、“密码学”或“隐私计算”等课程的配套数学基础教材，也可供读者自学使用。建议按章节顺序阅读，以建立循序渐进的知识体系。
